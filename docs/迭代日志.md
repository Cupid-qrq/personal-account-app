## v0.3 - 2026-04-01（科技感升级 + 数据洞察 + 部署指南）

新增/改进：

### UI/UX 升级（科技感）
- 黑色背景深色主题 + 渐变蓝紫配色
- 毛玻璃容器卡片（Glass Morphism 设计）
- 饼图放大至 2/3 屏幕宽度（原 1/2）
- 新型指标卡片：蓝紫渐变 + Hover 上浮动画
- Plotly 图表背景透明 + 深色专业配色
- 响应式布局优化，图表高度增加

### 数据分析深化
- 🧠 **消费洞察模块**：消费效率评分（0-100分）、消费指标分析
- ⚠️ **风险预警模块**：日均消费分析、异常支出检测、预警阈值
- 📈 **消费习惯分析**：总支出、日均消费、平均单笔、消费天数、最高频分类
- 💡 **智能建议模块**：基于评分的个性化改进建议（4-5条）
- 📊 **评分细项**：储蓄率、消费均衡度、分类多样性、数据完整度

### 页面结构调整
- 4 个分析标签页：支出分析 | 消费洞察 | 风险预警 | 详细账目
- 支出分析页：大饼图（占比提升） + 快捷分类查看按钮
- 新增"分类快捷按钮"：一键查看各分类详细数据
- 详细账目页：汇总 Top10、预算建议、原始明细

### 新增分析函数（src/analytics.py）
- `consumption_alerts(df)` - 日均分析、异常检测、预警列表
- `consumption_habit(df)` - 消费习惯指标
- `spending_efficiency_score(df)` - 综合评分（储蓄率、均衡度、多样性、完整度）
- `category_trend(df)` - 分类消费趋势（按日期和分类）

### 部署与分享
- 新增部署指南文档：5 种部署方案详解
- Streamlit Cloud 一键部署（推荐）
- Docker + 云服务器方案
- 本地 NAS 方案（家庭内网）
- Heroku 快速原型
- 安全建议与故障排查
