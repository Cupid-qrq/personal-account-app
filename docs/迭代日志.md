# 迭代日志

## v0.6 - 2026-04-05 (企业级升级・智能分析引擎)

**版本主题:** 五维度系统升级 - 认证 | 分析 | UI | 架构 | 工程化

### 🔐 认证系统全面升级 (src/auth.py)
- **RBAC 权限模型**: 从 admin/viewer 二元制升级到 7 权限细粒度控制
  - 新权限: `view_dashboard`, `view_analytics`, `view_details`, `export_data`, `upload_data`, `manage_users`, `system_settings`
  - 支持自定义角色组合
- **密码安全加强**: 硬编码 → SHA256+Salt 加密存储 + 环境变量读取
- **审计日志系统**: 新增 `AuditLogger` 类，记录所有登录/登出事件，支持查询历史审计
- **会话管理**: 为后续 2FA 和生物识别埋设接口

### 📊 分析引擎深度扩展 (src/analytics.py)
新增 5 大高级分析函数：
  1. **`year_over_year_comparison()`** - 年度对比分析，识别季节性规律
  2. **`expense_health_index()`** - 四维健康指数 (储蓄/稳定/控制/多元) + 分等级建议
  3. **`category_budget_forecast()`** - 分类预算预测，基于历史均值+波动范围
  4. **`anomaly_detection()`** - 异常交易检测 (超大金额 + 低频分类)
  5. **`generate_smart_insights()`** - AI 驱动洞察生成，综合多维指标输出可执行建议

### 🎨 UI/UX 体系现代化升级
- **视觉主题升级**: 深空主题 (0a141f) + 蓝橙配色 + 玻璃态设计
- **导入优化**: 新增 20+ 强调色、边框、阴影变量到 config 系统
- **交互设计预留**: 为悬停反馈、平滑过渡、骨架屏加载预留接口

### ⚙️ 项目配置中心化 (src/config.py)
- **配置集中管理**: 所有魔术数字 → 定量常数
- **功能开关系统**: 8 项功能开关，便于 AB 测试 + 灰度发布
- **颜色系统**: 12 色彩变量 + Streamlit 主题预设
- **性能配置**: 缓存有效期、显示行数、异常灵敏度等重数可配

### 🧪 导入与函数调用优化
- **app.py 导入**: 从 14 个函数 → 24 个（新增 10 个高级分析函数 + 4 个认证函数）
- **deps 扩展**: 新增 `numpy` 和 `python-dotenv` 依赖支持更复杂计算 + 环境变量管理

### 📋 迭代工程化决策
