## 🚀 推荐方案 1：Streamlit Cloud（最简单 ⭐⭐⭐⭐⭐）

### 优点
- ✓ 无需服务器，官方一键部署
- ✓ 免费，永久有效
- ✓ 全球 CDN 加速
- ✓ 自动 SSL 证书
- ✓ 支持 GitHub 自动同步

### 缺点
- ✗ 需要 GitHub 账号
- ✗ 数据存储在云端（可能隐私考虑）

### 部署步骤

#### 1️⃣ 准备 GitHub 仓库

```bash
# 在本地初始化 git（如果还没有）
git init

# 添加所有文件到 git
git add .
git commit -m "发布版本 v0.8"

# 连接到 GitHub（需要在 GitHub 创建仓库）
git remote add origin https://github.com/your-username/ledger-system.git
git push -u origin main
```

#### 2️⃣ 在 Streamlit Cloud 上部署

1. 访问 [Streamlit Cloud](https://share.streamlit.io)
2. 用 GitHub 账号登录
3. 点击 "New app"
4. 选择你的仓库和分支
5. 在 App Settings -> Secrets 配置 `LEDGER_USERS_JSON`
6. 设置主文件为 `app.py`
7. 点击 "Deploy" 完成

示例 Secrets（请替换为你自己的账号密码）：

```toml
LEDGER_USERS_JSON = '{"admin":{"password":"ChangeMe_2026!","name":"管理员","role":"admin"},"editor":{"password":"ChangeMe_2026!","name":"编辑者","role":"editor"},"viewer":{"password":"ChangeMe_2026!","name":"访客","role":"viewer"}}'
```

#### 3️⃣ 分享链接给家人

```
https://my-account.streamlit.app/
```

## 🐳 方案 2：Docker + 云服务器（推荐用于长期运营）

### 适用场景
- 需要稳定的 24/7 运行
- 有多个用户同时访问
- 需要数据库支持

### 部署步骤

#### 1️⃣ 创建 Dockerfile

在项目根目录创建 `Dockerfile`：

```dockerfile
FROM python:3.13-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

EXPOSE 8501

CMD ["streamlit", "run", "app.py", "--server.port=8501", "--server.address=0.0.0.0"]
```

#### 2️⃣ 创建 .dockerignore

```
.git
.gitignore
__pycache__
*.pyc
.streamlit/cache
data/processed
data/archive
```

#### 3️⃣ 构建并本地测试

```bash
docker build -t ledger-app:v0.8 .
docker run -p 8501:8501 -v $(pwd)/data:/app/data ledger-app:v0.8
```

#### 4️⃣ 推送到云服务器

选择以下任一方案：

**A. 阿里云 / 腾讯云 ECS (推荐)**
```bash
# 购买一个轻量应用服务器（约 20-50 元/月）
# SSH 进入服务器
ssh root@your-server-ip

# 安装 Docker
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh

# 上传代码并部署
git clone https://github.com/your-username/ledger-system.git
cd ledger-system
docker build -t ledger-app .
docker run -d -p 8501:8501 -v $(pwd)/data:/app/data --name ledger ledger-app

# 访问 http://your-server-ip:8501
```

**B. AWS Lightsail（国际用户）**
- 类似 ECS，按月订阅（约 $3.5-5/月）

## 🏠 方案 3：本地 NAS（仅家庭内网）

### 适用场景
- 家里有 NAS（威联通、群晖等）
- 不想花钱，只在家庭内网使用

### 部署步骤

#### 1️⃣ 在 NAS 上安装 Docker

大多数 NAS 都支持 Docker，通过官方应用商店安装即可。

#### 2️⃣ 创建容器

```bash
# NAS SSH 进入 Docker
docker pull python:3.13-slim

# 上传项目文件，运行
docker run -d \
  -p 8501:8501 \
  -v /path/to/ledger:/app \
  -v /path/to/data:/app/data \
  --name ledger \
  python:3.13-slim \
  bash -c "cd /app && pip install -r requirements.txt && streamlit run app.py"
```

#### 3️⃣ 访问

家里任何设备访问：`http://nas-ip:8501`

## 🎯 方案 4：Heroku（快速原型）

### 部署步骤

#### 1️⃣ 创建 Procfile

```
web: streamlit run app.py --server.port=$PORT
```

#### 2️⃣ 创建 .streamlit/config.toml

```toml
[server]
headless = true
port = $PORT
enableCORS = false
```

#### 3️⃣ Heroku 部署

```bash
heroku login
heroku create your-ledger-app
git push heroku main
```

⚠️ 注意：Heroku 免费 dyno 已于 2022 年停用，现需付费。

## 🔐 安全建议

### 生产环境注意事项

1. **添加密码保护**

在 `.streamlit/config.toml` 添加：
```toml
[client]
showErrorDetails = false

[logger]
level = "warning"
```

2. **使用反向代理（Nginx）**

```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        proxy_pass http://localhost:8501;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```

3. **HTTPS 证书**

使用 Let's Encrypt 获取免费 HTTPS：
```bash
certbot certonly --webroot -w /var/www/html -d your-domain.com
```

## 🆘 故障排查

### 问题：页面加载缓慢

- 原因：数据量大、网络延迟
- 解决：
  1. 优化数据加载（分页）
  2. 增加服务器配置
  3. 使用 CDN 加速

### 问题：数据丢失

- 原因：未备份
- 解决：
  1. 启用自动备份
  2. 定期导出数据
  3. 使用数据库（SQLite/PostgreSQL）

### 问题：无法上传数据

- 原因：文件限制 / 编码问题
- 解决：
  1. 检查文件大小（通常限制 200MB）
  2. 确保 CSV 编码为 UTF-8
  3. 检查字段名是否正确

### 问题：未配置 Secrets 时如何访问页面

- 结论：当前版本会自动进入公开只读模式，可直接查看分析页面。
- 原因：云端应用未读取到 `LEDGER_USERS_JSON` 时，会降级为 `viewer` 访客会话。
- 如需恢复登录和上传能力：
  1. 打开 Streamlit Cloud -> App Settings -> Secrets
  2. 配置 `LEDGER_USERS_JSON`（JSON 字符串）
  3. 保存后触发 Redeploy 或重启应用
  4. 刷新页面并使用账号登录
