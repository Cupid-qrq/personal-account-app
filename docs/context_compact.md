# Context Compact - 账本管理系统 v0.5

更新时间: 2026-04-02

## 项目目标
- 完成账单 CSV 的导入、清洗、去重、按月归档、主表汇总。
- 提供可视化分析与消费洞察，支持管理员上传、访客查看。
- 在移动端和桌面端保持可用的分析体验。

## 当前版本状态
- 版本: v0.5
- 主入口: app.py
- 分析模块: src/analytics.py
- 数据流水线: src/data_pipeline.py
- 认证模块: src/auth.py

## v0.5 核心更新
1. 修复金额重复显示: 概览指标卡不再二次拼接金额字符串。
2. 修复导入排序崩溃: 统一合并阶段时间列类型，避免 Timestamp/str 混排报错。
3. 修复预警边界异常: 无支出月份返回稳定 0 指标，避免 NaN。
4. 新增月度分析: 月度收入/支出/结余趋势、储蓄率、环比变化。
5. 完成 2026 年 1-2 月归档导入。

## 功能结构
- 登录与权限:
  - 管理员: 上传/导入 + 查看分析
  - 访客: 仅查看分析
- 主界面四大板块:
  - 支出分析
  - 消费洞察
  - 风险预警
  - 详细账目
- 新增月度对比区:
  - 多月趋势折线图
  - 当月储蓄率
  - 收入/支出/结余环比
  - 月度汇总表

## 迭代发布工作流（强制）
1. 完成功能或修复后，必须先执行基础校验（语法/关键数据检查）。
2. 校验通过后，必须执行 commit 并 push 到远程主分支。
3. 若线上连接 Streamlit Cloud，push 后检查自动部署状态；失败则手动 Redeploy。
4. 对外 README 仅保留用户导向信息，不暴露内部调试细节与演示凭证。
