# Context Compact - 账本管理系统

更新时间：2026-04 v0.4（多用户认证 + 移动端优化）

## 核心目标

- 对月度账单 CSV 进行导入、清洗、归档与高级可视化分析。
- 支持多用户认证：管理员上传数据，家人访客仅查看。
- 确保移动端体验流畅，自适应各屏幕尺寸。
- 部署到 Streamlit Cloud，全球用户无障碍访问。

## 已实现能力（v0.4）

### 🔐 身份认证模块（NEW）
- 登录页面：用户名 + 密码 + 演示账号提示
- 三账户体系：cupid(admin) + dad(viewer) + mom(viewer)  
- 权限粒度：admin=可上传 + 查看，viewer=仅查看
- 会话管理：`st.session_state` 存储用户身份，页面刷新保留状态
- 登出功能：侧边栏快速登出

### 📱 移动端完整适配（NEW）
- 响应式 CSS：`@media (max-width: 768px)` 和 `@media (max-width: 480px)`
- 单列布局：移动设备自动降级为 1 列（平板/PC 保持 2+ 列）
- 字体自适应：指标值/标签在移动端自动缩放、保持可读性
- 卡片自适应：Padding/高度/边距在小屏自动调整
- Plotly 图表：高度动态设定，防止超宽
- 触摸优化：按钮尺寸增大（>44px），点击区域友好

### 🎨 UI 清理与优化（NEW）
- 深色主题强制锁定：`.streamlit/config.toml` 配置 `theme.base="dark"` + 隐藏主题切换器
- 侧边栏重组织：用户信息 + 角色标注 + 权限说明
- 数据管理面板：仅 admin 可见，viewer 看到"您是访客账户"提示
- 按钮清理：仅保留 Star（⭐ GitHub）+ 跳转 Repo 链接
- 登录卡片美化：毛玻璃渐变 + 蓝紫配色 + 账户提示

### 前端设计（保留 v0.3）
- 深色主题（黑紫系）+ 毛玻璃容器卡片（Glass Morphism）
- 渐变蓝紫配色系统 + Hover 上浮动画  
- 大幅饼图：占屏幕 2/3 宽度、清晰标签显示
- 4 个分析标签页：支出分析 | 消费洞察 | 风险预警 | 详细账目
- 分类快捷按钮：一键钻取分类明细

### 数据分析深化（保留 v0.3）
- **消费效率评分**（0-100）：储蓄率40% + 消费均衡度20% + 分类多样性20% + 数据完整度20%
- **风险预警**：日均消费分析、日消费波动度、异常支出检测（阈值 = 日均 + 1σ）
- **消费习惯**：总支出、日均消费、平均单笔、消费天数、最高频分类
- **智能建议**：基于评分的个性化改进建议（4-5条）
- **预算建议**：基于本月支出的下月预算提议

- `app.py` - 主应用（包含登录 + 权限检查 + 响应式布局）【v0.4重写】
- `src/auth.py` - 认证模块（USERS_DB、authenticate_user、can_upload）【v0.4新增】

### 技术栈
- 核心：Python 3.13 + Streamlit 1.56 + Pandas 2.2+ + Plotly 6.6
- 认证：自定义 session-based（无第三方依赖）
- 部署：GitHub + Streamlit Cloud（自动 webhook 同步）
- 样式：HTML/CSS 深色毛玻璃设计
- 部署：支持 Docker、Heroku、Streamlit Cloud

## 关键设计决策

- **深色主题**：减少眼疲劳，更科技感
- **毛玻璃卡片**：现代 UI 审美
- **表单化标签页**：清晰信息分层
- **数据驱动评分**：量化消费健康度
- **预警阈值**：标准差法则实现智能异常检测
- **Session State**：记录用户交互状态
- **多部署方案**：满足不同场景需求
