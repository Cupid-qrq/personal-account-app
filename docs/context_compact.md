# Context Compact - 账本管理系统 v0.5 Final

更新时间: 2026-04-02

## 项目目标
- 完成账单 CSV 的导入、清洗、去重、按月归档、主表汇总。
- 提供可视化分析与消费洞察，支持管理员上传、访客查看。
- 在移动端和桌面端保持可用的分析体验。

## 当前版本状态
- 版本: v0.5 Final（封版）
- 主入口: app.py
- 分析模块: src/analytics.py
- 数据流水线: src/data_pipeline.py
- 认证模块: src/auth.py

## v0.5+ 核心更新
1. 视觉层升级：动态背景、浮动光斑、3D 质感指标卡、Hero 信息区。
2. 月度分析升级为四维中心：趋势总览、结构透视、消费节律、智能洞察。
3. 新增数据函数：`monthly_category_share`、`monthly_rhythm_heatmap`、`monthly_insight_digest`。
4. 保留并延续 v0.5 的稳定性修复与数据归档成果。

## v0.5 核心更新
1. 修复金额重复显示: 概览指标卡不再二次拼接金额字符串。
2. 修复导入排序崩溃: 统一合并阶段时间列类型，避免 Timestamp/str 混排报错。
3. 修复预警边界异常: 无支出月份返回稳定 0 指标，避免 NaN。
4. 新增月度分析: 月度收入/支出/结余趋势、储蓄率、环比变化。
5. 完成 2026 年 1-2 月归档导入。

## 功能结构
- 登录与权限:
  - 管理员: 上传/导入 + 查看分析
  - 访客: 仅查看分析
- 主界面四大板块:
  - 支出分析
  - 消费洞察
  - 风险预警
  - 详细账目
- 新增月度洞察中心:
  - 趋势总览（多月趋势 + 环比）
  - 结构透视（占比堆叠 + 3D 结构）
  - 消费节律（周序 x 周几热力）
  - 智能洞察（排名 + 集中度 + 波动结论）
